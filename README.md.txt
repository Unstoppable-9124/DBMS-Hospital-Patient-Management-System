# 🏥 Hospital Management System

## 📌 Overview

This project is a Hospital Management System built using Python and MySQL. It allows managing patient records, doctors, and appointments.

## 🛠️ Technologies Used

* Python
* MySQL

## ⚙️ How to Run

1. Import database:

   ```sql
   source hospital.sql;
   ```
2. Run the project:

   ```bash
   python app.py
   ```

## 📂 Files

* app.py → Main application
* db.py → Database connection
* hospital.sql → Database structure and data

## 🚀 Features

* Add Patients
* View Records
* Manage Appointments
